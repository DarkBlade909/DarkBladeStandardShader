# DarkBlade Standard Shader
Basic Standard shader for VRChat with some extra features.

## REQUIREMENTS
- z3y's Graphlit: https://github.com/z3y/Graphlit
## Features
- Packed ORM Maps
- Detail Maps
- Clearcoat
- Anisotropy
- Cloth shading
- Retroreflective shading
- Geometric Specular Anti-aliasing
- Indirect Specular
- Specular Occlusion
- Flipped Backface Normals
- Seperate Backface/UV1 AO
- Shadow Mask
- VRC Light Volumes
